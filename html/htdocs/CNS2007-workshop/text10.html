Notes:

- Three main roles

- Initial get software
- Initiate a research project

- Push copies the whole project, including dead ends to the publication database
- Pushes can be done incrementally

- Pull for reading a paper
- Pull can be done in a narrative mode (story only)
- Pull can be done in a full project mode (includes dead ends)

- Database contains everything, including all dead ends of all projects
- Database is 'meta analysis enabled', 'model mining'

- Optionally during the research, collaborators can pull and push. This feature is under consideration, but unlikely for the first release
