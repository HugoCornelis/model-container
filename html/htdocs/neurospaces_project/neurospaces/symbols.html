CachedConnection (cached_connection)

a connection in a projection index, alien typed and with alien identification numbers

Variables and Parameters:

DELAY
connection delay
POST
connection post synaptic serial identifier
PRE
connection pre synaptic serial identifier
WEIGHT
connection weight

Channel (channel)

a ion conductance through a cell membrane

Variables and Parameters:

CHANNEL_TYPE
defines the channel type (ChannelActInact: channel with activation and inactivation gates, ChannelAct: channel with activation gate, ChannelActConc: channel with activation and concentration gates)
Erev
channel reversal potential
G
actual channel conductance, normally a solved variable
GMAX
maximal conductance density when all channels are in the open state
I
actual channel current, normally a solved variable

ConcentrationGateKinetic (concentration_gate_kinetic)

a description of the kinetics of a channel gate that is concentration dependent

Variables and Parameters:

Base
basal level, A in EDS1994
Tau
time constant, B in EDS1994

ContourPoint (contour_point)

Equation (equation)

Fiber (fiber)

Variables and Parameters:

RATE
average firing frequence, bind it to the threshold of a spikegenerator
REFRACTORY

GateKinetic (gate_kinetic)

a description of the kinetics of a gate, only in the forward or backward direction

Variables and Parameters:

HH_Factor_Flag
3: choose between nominator or denominator, 1 means nominator, -1
HH_Mult
2: multiplier membrane dependence, 0.0 for no dependence
HH_Offset
4: nominator or denominator offset
HH_Offset1
4: nominator or denominator offset
HH_Offset2
5: membrane offset
HH_Scale
1: multiplier
HH_Tau
6: denormalized time constant

HHGate (h_h_gate)

a channel gate with hodgkin huxley alike kinetics

Variables and Parameters:

POWER
gate power
state
current state, normally a solved variable
state_init
initial value, commonly forward over backward steady states

Pool (pool)

a concentration pool, exponential decay

Variables and Parameters:

BASE
base concentration level
BETA
fixed beta (use the FIXED function to avoid scaling)
DIA
concentration pool diameter, normally the same as the segment diameter
LENGTH
concentration pool length, normally the same as the segment length
TAU
concentration pool time constant
THICK
concentration pool thickness
VAL
ion valency
concen
actual concentration value, normally a solved variable
concen_init
initial concentration level

Randomvalue (randomvalue)

Variables and Parameters:

LENGTH
length, for visualization purposes only
MAXIMUM

MINIMUM
minimum value the random variable can take

Segment (segment)

a single linear cylindrical segment of a neuron

Variables and Parameters:

BRANCHPOINT
1 if this is a branch point, 0 if not, FLT_MAX if unknown (a very high value)
CM
specific capacitance of this segment
RA
specific axial resistance of this segment
RM
specific membrane resistance of this segment
SOMATOPETAL_BRANCHPOINTS
number of branchpoints on the path toward the soma
SOMATOPETAL_DISTANCE
somatopetal distance of the segment
TAU
time constant of the segment, CM * RM
Vm
membrane potential, normally a solved variable
Vm_init
initial membrane potential

Segmenter (segmenter)

a cable based component that allows segmentation of self or/and its children

Variables and Parameters:

DIA
segment diameter
LENGTH
segment length (calculated automatically if not set)
PARENT
parent segment, somatopetal
SURFACE
surface of the segment (calculated automatically if not set, you can set it to add spine surface)
TOTALSURFACE
total surface of all segments in the segmenter (calculated automatically)
rel_X
spatial extent of the segment in the X direction
rel_Y
spatial extent of the segment in the Y direction
rel_Z
spatial extent of the segment in the Z direction
