This example is one of the regression tests of the Neurospaces project.

DB interface fetches morphology from the neuromorpho DB, over the internet.

Modelcontainer stores the morphology / model, and makes it available for other software components.

The project browser and studio activate, per user request, different algorithms in the model container, that compute important properties of the morphology, and are presented to the user.

Allows to extract surface area, number of branch points, branch order, etc. all important computational properties of a morphology.
